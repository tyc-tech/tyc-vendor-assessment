# Kraljic 矩阵 · 供应商分类策略

## 理论基础

Kraljic 矩阵（Peter Kraljic, 1983）按两维度将供应商分为 4 象限，对应不同采购策略。

```
                    采购风险 / 供应难度
                    ─────────────────────►
                    低                   高
                    ┌────────────┬────────────┐
       高     价值  │            │            │
        ↑          │   杠杆      │   战略      │
        │  采      │ Leverage   │ Strategic  │
        │  购      ├────────────┼────────────┤
        │  价      │            │            │
        │  值      │   一般      │   瓶颈      │
       低          │ Routine    │ Bottleneck │
                    └────────────┴────────────┘
```

## TYC 的自动落位算法

基于 9 维度评估结果，按以下映射自动落位：

| 维度 | 采购价值信号 | 采购风险信号 |
|------|------------|------------|
| ① 工商资质 | 规模 + 存续 | — |
| ② 股权 | — | 实控不清 = 风险高 |
| ③ 财务 | 营收 + 利润 | 负债率 = 风险 |
| ④ 司法 | — | **主要风险来源** |
| ⑤ 经营异常 | — | **主要风险来源** |
| ⑥ 税务 | — | 税务违法 = 风险 |
| ⑦ 知识产权 | 专利实力 = 高价值 | — |
| ⑧ 招投标 | 中标记录 | — |
| ⑨ 供应链 | 产能稳定 = 价值 | 单一来源 = 风险 |

采购价值分 = (③ + ⑦ + ⑧) × 33%
采购风险分 = (② + ④ + ⑤ + ⑥ + ⑨的风险侧) × 20%

## 4 象限策略

### 战略（Strategic）· 高价值 + 高风险

- 长期合作协议（3-5 年）
- 联合研发 / VMI（供应商管理库存）
- 风险缓冲（多源 + 备选 + 保险）
- 深度绑定（交叉持股 / 联合招聘）

### 杠杆（Leverage）· 高价值 + 低风险

- 多源竞标（≥3 家）
- 议价谈判
- 集中招标 / RFQ 策略
- 短期合同 + 频繁复评

### 瓶颈（Bottleneck）· 低价值 + 高风险

- 备用库存 / 安全库存
- 开发备选供应商
- 合同保障（交付 / 质量 / 价格）
- 签长期供货协议锁价

### 一般（Routine）· 低价值 + 低风险

- 集中采购节约
- 标准化 / 电商渠道
- 自动续约
- 最小化管理投入
